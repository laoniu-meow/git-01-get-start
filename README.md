# Git for beginner
In this topic, the author has developed a set of Git guidelines for beginners, covering account setup, basic configuration, and how to commit projects to GitHub. While some of these guidelines are already included in the author’s existing GitHub repositories, the intention is to create a dedicated repository focused entirely on Git-related topics.

This approach aims to provide a centralised learning resource that is easy to follow and continuously improve. The author also encourages contributions from others to share additional knowledge, best practices, and insights, fostering collaborative learning and supporting skill development for everyone.
